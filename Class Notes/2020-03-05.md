## Possession Pronoun

| Personal | Choice     | Example                                              |
| -------- | ---------- | ---------------------------------------------------- |
| ich      | mein(e)    | **mein** Stift / Mann, **meine** Frau, **mein** Kind |
| du       | dein(e)    | **dein** Stift / Mann, **deine** Frau, **dein** Kind |
| Sie      | **I**hr(e) | **Ihr** Stift / Mann, **Ihre** Frau, **Ihr** Kind    |
