## Verben mit <ruby>Präfix<rt>prefix</rt></ruby>

| Type                                      | Choice               | Examples                                                     |
| ----------------------------------------- | -------------------- | ------------------------------------------------------------ |
| be-, ver-, emp-, ent-, her- miss- ge- er- | split                | **be**zahlen: Ich bezahle bar.                               |
| unter-, über-, wider-, um-                | some split, some not | **<u>wieder</u>**kommen: Ich komme morgen wieder.<br />**wieder**<u>holen</u>: Ich wiederhole die Grammatik. |
| All others                                | not split            | **auf**stehen: Ich steht um 9.00 Uhr auf.                    |

> General rule: If the emphasis is **AFTER** the prefix, it should **NOT** be split.
